## 🎯 Overview

Decode tattoo ink pigment names, CAS numbers, and CI numbers against EU Regulation 2020/2081. Identify banned and restricted ingredients fast.

**Category:** Tattoo Engineering

This is a free, open-source widget from the [Poli International Widget Suite](https://poliinternational.com/tools/) - a collection of practical tools for tattoo and piercing studios, artists, and clients. It runs entirely in the browser; no data is sent to a server.

---

## ✨ Features

- ✅ **Tattoo Ink Ingredients**
- ✅ **EU 2020/2081**
- ✅ **Pigment Blue 15 Banned**
- ✅ **Restricted Pigments**
- ✅ **CI Number Check**
- ✅ **Tattoo Ink Compliance**
- ✅ **Responsive Design** - works on desktop, tablet, and mobile.
- ✅ **Free & Open Source** - MIT licensed.

## 🚀 Installation

### Option 1: Use Online (Recommended)

No installation needed! Just visit:
**[https://poliinternational.com/tools/ink-ingredient-decoder/](https://poliinternational.com/tools/ink-ingredient-decoder/)**

### Option 2: Run Locally

This tool is pure HTML/CSS/JS - no build step. Clone the repo and open `index.html`:

```bash
git clone https://github.com/Poli-International/ink-ingredient-decoder.git
cd ink-ingredient-decoder
# open index.html in your browser
```
